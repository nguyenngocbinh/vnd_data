# vnstock

vnstock is a Python library for retrieving historical stock data from the VNDIRECT API.

## Installation

You can install vnstock using `pip`:

```
pip install vnstock
```


## Usage

```python
import vnstock

tickers = ['TPB', 'VCB', 'HCM']
size = 100
df = my_package.get_data(tickers, size)
print(df)
```
